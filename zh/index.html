常见问题

如何创建账户？

打开应用，点击"开始使用"。您可以使用电子邮件或 Apple ID 注册。

如何恢复我的订阅？

前往 设置 → 账户 → 恢复购买。请确保使用与原始购买相同的 Apple ID 登录。

如何删除我的账户？

前往 设置 → 账户 → 删除账户。请注意，此操作不可逆。

应用无法正常工作怎么办？

尝试关闭并重新打开应用。如果问题仍然存在，请联系我们并提供您的设备信息和遇到的问题详情。
